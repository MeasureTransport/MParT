Installation#

Install from Conda

COMING SOON!

conda install -c conda-forge mpart

Compiling from Source

MParT is configured with CMake and a basic compilation can be achieve with

mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=<some path> ..
make install
