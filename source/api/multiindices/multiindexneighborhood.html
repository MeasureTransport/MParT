class mpart::DefaultNeighborhood : public mpart::MultiIndexNeighborhood¶

Defines the standard graph connectivity for a MultiIndexSet

.

A multiindex

\(\mathbf{\alpha}\) is a forward neighbor of \(\mathbf{\beta}\) if \(\|\mathbf{\alpha}-\mathbf{\beta}|_1==1\) and \(\mathbf{\alpha}_i \geq \mathbf{\alpha}_i\) for all \(i\).

A multiindex \(\mathbf{\alpha}\) is a backward neighbor of \(\mathbf{\beta}\) if \(\|\mathbf{\alpha}-\mathbf{\beta}|_1==1\) and \(\mathbf{\alpha}_i \leq \mathbf{\alpha}_i\) for all \(i\).
