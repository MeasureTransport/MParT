Clenshaw Curtis Quadrature#

class mpart::ClenshawCurtisQuadrature#

USAGE 

ClenshawCurtisQuadrature quad(5);
integral = quad.Integrate(f, 0,1);


Eigen::VectorXd wts, pts;
std::tie(wts, pts) = ClenshawCurtisQuadrature::GetRule(order);

Public Functions

inline ClenshawCurtisQuadrature(unsigned int order)#
template<class FunctionType>
inline auto Integrate(FunctionType const &f, double lb, double ub) -> decltype(f(0.0))#

Approximates the integral \(\int_{x_L}^{x_U} f(x) dx\) using a Clenshaw-Curtis quadrature rule.

Parameters:

  • f – [in] The integrand. Can return any type that overloads multiplication with a double and the += operator. doubles and Eigen::VectorXd are examples.

  • lb – [in] The lower bound \(x_L\) in the integration.

  • lb – [in] The upper bound \(x_U\) in the integration.

Template Parameters:

FunctionType – The type of the integrand. Must have an operator()(double x) function.

Returns:

An approximation of \(\int_{x_L}^{x_U} f(x) dx\). The return type will be the same as the type returned by the integrand function.
